🚀 Publicar no GitHub Pages
O projeto está pronto para ser publicado direto deste repositório.
Passo a passo:
  1. Faça git push origin main do repositório local.
  2. No GitHub abra o repositório, clique em Settings.
  3. Na lateral esquerda, abra Pages.
  4. Em Source selecione Deploy from a branch.
  5. Em Branch, escolha main + pasta / (root). Salve.
  6. Aguarde ~2 minutos. O link público fica disponível em https://seu-usuario.github.io/flowtrack.
💡 Dica: Compartilhe apenas o link com seu time. Cadastre os e-mails deles em Usuários para restringir o acesso.
